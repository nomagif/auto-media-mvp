# AI_MACRO_SOURCE_EXPANSION_PLAN

AI・テクノロジー / 金融・経済（投資・マクロ）軸で observatory を広げるための source expansion plan。

## 1. 方針
今後の source 追加は、原則として以下のどちらかに明確に寄与するものを優先する。

- **AI / Technology**
- **Finance / Economics (Investing / Macro)**

補助的に、以下の cross-cutting lens を重視する。
- semiconductors
- policy / regulation
- security
- crypto / market-data

## 2. 優先 source map

### A. AI / Technology
#### Tier 1
- TechCrunch AI / startup coverage
- Hacker News (technology / infra / open source discovery)
- company engineering / research blogs

#### Tier 2
- Anthropic news / announcements
- OpenAI news / announcements
- Google / DeepMind blog
- Nvidia blog / newsroom

#### Tier 3
- semiconductor industry feeds
- cloud / developer infra blogs
- selected security research feeds

### B. Finance / Economics (Investing / Macro)
#### Tier 1
- Federal Reserve / FOMC
- CoinGecko market snapshots

#### Tier 2
- U.S. Treasury / BLS / BEA / SEC public releases
- ETF / crypto policy sources
- major exchange / market structure public updates

#### Tier 3
- ECB / BOE / BOJ public releases
- commodity / rates / FX related public data feeds

## 3. 次の有力候補
### 候補1: OpenAI / Anthropic / Google announcements
狙い:
- AI observatory としての主題を強める
- product-launch / research / infrastructure を厚くする

### 候補2: U.S. Treasury / BLS / BEA
狙い:
- macro coverage を Fed 以外にも広げる
- inflation / labor / growth の signal を増やす

### 候補3: semiconductor source
狙い:
- AI と macro の接点を作る
- chips / supply chain / capex を拾いやすくする

## 4. おすすめの実装順
1. AI official source を 1 本追加
   - OpenAI or Anthropic
2. macro public source を 1 本追加
   - BLS or Treasury
3. semiconductor source を 1 本追加
4. その後に quality tuning

## 5. 判断基準
新しい source は以下で評価する。
- 主ジャンルに明確に寄与するか
- recurring collect に向いているか
- raw / normalized 化が簡単か
- ranking に入れたとき general を増やしすぎないか
- public / low-risk source か

## 6. 率直なおすすめ
次の 1 本は **AI official source** がよい。

理由:
- いま macro 側は Fed と CoinGecko で足場ができた
- AI 側も official / primary source を入れるとバランスが良くなる
- observatory の柱がより明確になる
