Links expire after 15 minutes. You can always come back here and issue fresh ones.

Success means: your license verifies, signed links appear below, and the linked file downloads from Cloudflare R2.

If the premium API is hosted on a separate Worker, set its base URL here once and the page will remember it in this browser.

    How this works

    • The access form verifies your Gumroad license key server-side.
    • If the license is valid, the site generates short-lived signed links for the latest files.
    • Premium files are expected to live in Cloudflare R2, not in the public Git repo.

    If you lost your license key, use Gumroad’s receipt email or account page first.

    Buyer checklist

    • Buy on Gumroad and copy the license key from your receipt or library.
    • Pick the matching product on this page.
    • Paste the same purchase email if available.
    • Confirm that download links appear and that at least one file opens successfully.

    If Gumroad purchase succeeds but this page fails, that means the commerce flow works and the self-serve license verification flow still needs debugging.

    Fastest E2E check

    1. Open your Gumroad receipt or library and copy the license key.
    2. Come back here and choose the matching product.
    3. Paste the purchase email and license key, then press Get latest download links.
    4. If you see signed links below, open one immediately and confirm the file downloads.

    Shortest success criterion: verified license → signed link appears → file downloads.
